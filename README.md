## How We Work

Every project follows the same 4-step process — so you always know what to expect.

```
01  Discovery & Planning  →  Requirements, architecture, full documentation before a line is written
02  Design & Prototype    →  High-fidelity designs and clickable prototype to validate before dev
03  Build & Iterate       →  2-week sprints, end-of-sprint demos, shared project board
04  Deploy & Support      →  Production deployment with post-launch monitoring and support included
```

**8–12 week MVPs.** Clear timeline on day one — and we stick to it.
